# 🎉 Try These With Your New AI Assistant

Copy any of the prompts below and run:

```
v1claw agent -m "YOUR PROMPT HERE"
```

Or start an interactive session and type them directly:

```
v1claw agent
```

---

## 👋 Getting Acquainted

```
Who are you and what can you help me with?
```

```
What tools do you have available?
```

---

## 🌐 Web Search

*(Requires DuckDuckGo enabled — run `v1claw configure` → Tools if not done yet)*

```
Search the web: what are the top AI assistants available today?
```

```
Find the current weather in New York City
```
